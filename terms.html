En téléchargeant et en utilisant LESS, vous acceptez les présentes conditions d'utilisation. Si vous n'acceptez pas ces conditions, veuillez ne pas utiliser l'application.

2. Description du service

LESS est une application de bien-être numérique qui aide les utilisateurs à réduire leur temps d'écran via la gamification, le blocage d'applications et des exercices de respiration. L'application utilise les APIs Screen Time d'Apple (FamilyControls, DeviceActivity, ManagedSettings).

3. Compte et utilisation

L'application fonctionne avec une authentification anonyme. Aucun compte avec email ou mot de passe n'est requis. Les données sont stockées localement sur votre appareil et dans Firebase pour les fonctionnalités sociales (défis entre amis).

4. Abonnement LESS PRO

LESS propose un abonnement premium (LESS PRO) avec des fonctionnalités avancées. Les prix sont affichés dans l'application. Le paiement est traité via l'App Store d'Apple. L'abonnement se renouvelle automatiquement sauf annulation au moins 24 heures avant la fin de la période en cours. Vous pouvez gérer votre abonnement dans les réglages de l'App Store.

5. Blocage d'applications

Le blocage d'applications est réalisé via les APIs FamilyControls d'Apple. LESS ne peut pas garantir un blocage à 100% dans toutes les situations. L'utilisateur reste responsable de son utilisation de son appareil.

6. Données et confidentialité

LESS ne collecte pas de données personnelles. Les données de temps d'écran restent sur votre appareil conformément aux règles d'Apple. Les données de défis entre amis sont stockées dans Firebase. Consultez notre politique de confidentialité pour plus de détails.
